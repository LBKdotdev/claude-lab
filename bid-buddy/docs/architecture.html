Data Flow: Inventory Sync

1
User taps "Sync Inventory"
HomeScreen.tsx triggers handleSync()
2
Supabase Edge Function
fetch-npa-inventory proxies to GCP Cloud Run
3
NPA API returns AuctionItem[]
year, make, model, VIN, score, mileage, docs
4
Map to InventoryItem, preserve status
Existing buddyTag/notes/maxBid kept on re-sync
5
Save to IndexedDB
lbk-bid-buddy store, indexed by itemNumber

Data Flow: Comps Lookup

1
Check LocalStorage cache
comps-v3-{query}, 30min expiry
2
Cache miss? Call fetch-comps
Supabase edge function (POST { query })
3
Parallel scraping
eBay sold + CycleTrader + 4x Craigslist cities
4
Aggregate & cache results
Sort by price, variant filter, never cache empty
